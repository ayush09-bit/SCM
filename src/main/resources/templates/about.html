About Secure Contact Manager

Secure Contact Manager is a modern web application built by Ayush Chaudhary, a B.Tech Computer Science student. The system helps users store, organize, and manage personal & professional contacts in a clean and efficient way.

This platform allows users to add, view, update, delete, and categorize contacts. Designed with a clean UI, responsive layout, and dark mode support, Secure Contact Manager focuses on usability, smooth navigation, and privacy-friendly contact storage.
